# Semgrep Demo Placeholder

This is a placeholder repository for demonstrating Semgrep integration.

**Note**: This repository contains intentional security vulnerabilities for demonstration purposes only.

## Vulnerabilities Included

- SQL Injection
- Cross-Site Scripting (XSS)
- Hardcoded Secrets
- Command Injection
- Weak Cryptography (MD5)

## Purpose

This repository serves as a demo tool to show:
- The difference between integrated and non-integrated repositories
- How to onboard a new repository to Semgrep
- The scanning process and findings
